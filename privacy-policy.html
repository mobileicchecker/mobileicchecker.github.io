MIC App Privacy Policy

MIC App values and protects your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our application.

Information We Collect

We use several Firebase services that automatically collect certain types of data. This may include:

1. Firebase Analytics

Collects general usage data such as:

  • Device information

  • App usage events

  • Session data

  • Country/region information

We use this data to improve app performance and user experience.

2. Firebase Cloud Messaging (Push Notifications)

FCM may collect:

  • Device token

  • Notification delivery information

Used only for sending app notifications.

3. Firebase In-App Messaging

Collects:

  • App events

  • Device information

  • Whether the user opened or dismissed a message

Used to show in-app messages.

4. Firebase Remote Config

Collects:

  • App version

  • Device information

  • A/B testing events

Used for dynamic updates inside the app.

5. Firebase Firestore Database

Our application uses Firestore only to read non-personal data, such as IC codes and related information.

  • We do not upload, store, or save any user personal data in Firestore.

  • Firestore is used strictly as a read-only database to fetch app-related information.

  • No user device information, login information, or identifiers are stored in Firestore.

We do not store unnecessary personal information.

6. Firebase Crashlytics

Collects:

  • Crash logs

  • Device information

  • App version

  • Stack traces

Used only to fix crashes and improve stability.

7. Firebase App Check

Collects verification information to ensure your app is used securely.

How We Use Your Information

We use the collected information to:

  • Improve app performance

  • Diagnose app crashes

  • Personalize user experience

  • Maintain app security

  • Deliver important notifications and messages

  • Update features dynamically

We never sell or rent your data to anyone.

Third-Party Sharing

Your data may be processed by:

  • Google Firebase services

  • Google Analytics for Firebase

  • Firebase Crashlytics

We do not share data with any other third parties.
