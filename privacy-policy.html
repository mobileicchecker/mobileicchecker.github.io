MIC App Privacy Policy

MIC App values and protects your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our application.

Information We Collect

We use several Firebase and Google services that automatically collect certain types of data. This may include:

1. Firebase Analytics

Collects general usage data such as:

  • Device information
  • App usage events
  • Session data
  • Country/region information

We use this data to improve app performance and user experience.

2. Firebase Cloud Messaging (Push Notifications)

FCM may collect:

  • Device token
  • Notification delivery information

Used only for sending app notifications.

3. Firebase In-App Messaging

  • App events
  • Device information
  • User interaction with messages

Used to show in-app messages.

4. Firebase Remote Config

  • App version
  • Device information
  • A/B testing events

Used for dynamic updates inside the app.

5. Firebase Firestore Database

Our application uses Firestore only to read non-personal data, such as IC codes and related information.

  • We do not upload or store personal user data in Firestore.
  • Firestore is used strictly as a read-only database.
  • No user identifiers or personal data are stored.

6. Firebase Crashlytics

  • Crash logs
  • Device information
  • App version
  • Stack traces

Used only to fix crashes and improve app stability.

7. Firebase App Check

Used to ensure secure and valid app usage.

Advertising

We use Google AdMob to display advertisements in our app. AdMob may collect and use certain data to provide better ad experiences, including:

  • Device identifiers (such as Advertising ID)
  • IP address
  • App interaction data

This data is used to show personalized and non-personalized ads. Users can control or reset their advertising preferences from their device settings.

How We Use Your Information

  • Improve app performance
  • Diagnose crashes
  • Enhance user experience
  • Maintain security
  • Deliver notifications
